# Winter Portfolio Website

A responsive portfolio website showcasing winter photography. This website features a modern design with a gallery, about section, contact form, and footer, optimized for all screen sizes.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile screens.
- **Hero Section**: Background slideshow with overlay and centered text.
- **Gallery**: Grid layout with support for images spanning multiple rows and columns.
- **Filter Links**: Category-based gallery navigation (All, Landscapes, Decoration, Food, Family).
- **About Section**: Two-column layout with text and images; includes cards highlighting experience and achievements.
- **Contact Form**: Collects user name, email, and message. Submit button spans full width.
- **Footer**: Simple footer with copyright and designer credit.

---

## Technologies Used

- **HTML5**: Markup structure.
- **CSS3**: Styling, Flexbox, and Grid layouts.
- **Responsive Design**: Media queries for screens below 1200px, 900px, and 650px.
